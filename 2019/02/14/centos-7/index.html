1丶在/etc/yum.repos.d/目录下新建virtualbox.repo并写入https://download.virtualbox.org/virtualbox/rpm/el/virtualbox.repo中的内容

2丶更新yum缓存，命令为yum clean all和yum makecache
3丶yum install virtualbox
